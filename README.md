# MoodVibez - Song Recommender 🎵

A song recommendation app that suggests music to you according to your mood,

## ✨ Features

- The user selects a mood and it recommends matching songs
- Simple and beautiful user interface
- Fast and accurate recommendations

## 🛠️ Tech Stack

- **MongoDB**
- **Express** 
- **React**
- **Node.JS**
- **Vite** – Fast development and build tool
- **Tailwind CSS** 
- **DaisyUI**
- **JWT** – for secure authentication
- **bcryptjs** – for password hashing
- **CORS & dotenv** – for security and environment variables
- **Google Generative AI** (@google/genai) – for smart music recommendations
- **Axios** – For making API requests
- **React Router** – For navigation
- **React Hot Toast** – For notifications
